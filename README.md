# daily-tech-digest

《Daily Tech Digest》是一个 **AI-first，但不是 AI-only** 的高价值综合科技日报项目。

## 它想解决什么问题
不是机械堆链接，也不是只盯模型发布，而是每天帮读者快速判断：
- 今天科技世界真正重要的变化是什么
- 哪些 AI、产品、工作流、平台和行业动态值得优先关注
- 哪些内容对普通人、小团队、创作者和操盘手真正有用

## 这份日报的价值
- **AI-first**：AI 仍然是主轴，但不缩成单一模型发布清单
- **综合科技视角**：覆盖 AI 应用、AI coding、平台、开源、基础设施、商业格局等
- **编辑判断**：强调「为什么重要」，而不是只搬运标题
- **可长期阅读**：既适合每天快速浏览，也适合做历史归档与趋势回看
- **可分享**：GitHub Pages 作为公开承载层，Discord 作为分发层

## 目录说明
- `index.html`：最新一期《Daily Tech Digest》
- `archive/index.html`：历史归档页
- `archive/YYYY-MM-DD.html`：往期日报页面
